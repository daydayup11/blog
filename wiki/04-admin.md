# 04 后台使用指南

后台地址：`http://localhost:8080/admin/index.html`

默认账号：`admin` / `admin123`

## 登录

输入用户名密码后，系统会从后端获取一个 JWT token 并保存在浏览器 localStorage 里。之后所有管理操作都会自动带上这个 token，无需重复登录。Token 有效期 7 天，到期后会自动跳回登录页。

---

## 统计面板

路径：`/admin/dashboard.html`

- **时间范围**：可切换今日 / 7天 / 30天
- **PV/UV 图表**：每日访问量柱状图（绿色=PV，蓝色=UV）
- **热门文章 Top 10**：按浏览人次排序
- **页面访问明细**：各路径的 PV/UV 明细表

> PV（Page View）= 总访问次数；UV（Unique Visitor）= 去重访客数（基于 IP+浏览器指纹）

---

## 文章管理

路径：`/admin/posts.html`

### 新建文章

点击右上角「+ 新建文章」进入编辑器。

### 编辑器说明

- **左侧**：Markdown 原文输入框
- **右侧**：实时预览（400ms 防抖，停止输入后自动更新）
- **导入 MD**：点击「📄 导入 MD」可从本地导入 `.md` 文件，内容会填入左侧编辑框
- **插入图片**：点击「📎 插入图片」上传图片，自动生成 `![name](url)` 语法插入光标处

### 保存 vs 发布

- **保存草稿**：文章存入数据库但不对外显示，后台文章列表可见
- **保存并发布**：保存后立即对外可见，出现在博客列表页

### 文章列表操作

| 按钮 | 说明 |
|---|---|
| 编辑 | 进入编辑器，加载已有内容 |
| 置顶/取消置顶 | 置顶文章显示在列表最前，有绿色 📌 标识 |
| 发布/下线 | 切换文章可见性 |
| 删除 | 永久删除，不可恢复 |

## 专栏管理

路径：`/admin/columns.html`

专栏是有序的文章集合，适合连载系列文章。

### 创建专栏

点击「+ 新建专栏」，输入专栏名称，系统自动生成唯一标识符（slug）。

### 管理专栏文章

点击「管理文章」按钮，弹出抽屉面板：

- **左侧**：当前专栏已有文章，拖拽 `⠿` 图标调整顺序
- **右侧**：所有文章列表，支持搜索，点「+ 添加」加入专栏
- **× 按钮**：将文章从专栏移出

点击「保存顺序」提交。

---

## 项目管理

路径：`/admin/projects.html`

### 项目 Tab

填写项目信息：
- **项目名**：显示名称
- **Stars**：GitHub Star 数
- **GitHub URL**：仓库链接
- **Demo URL**：演示地址（支持以下格式）
  - B站视频链接（自动嵌入播放器）
  - 图片地址（`.png/.jpg/.webp` 等）
  - 自部署服务地址（iframe 嵌入）
  - PDF 文件（embed 展示）
- **精选项目**：勾选后显示在主页「精选项目」区域

### 自定义板块 Tab

在作品集页面的「系统演示」和其他固定 Tab 之外，可以添加任意自定义 Tab：

1. 点击「+ 新建板块」，输入板块名（如"开源贡献"）
2. 板块创建后点「+ 添加条目」，可填写标题、链接、简介、嵌入地址
3. 前台作品集页面会自动出现新的 Tab

## 知识花园管理

路径：`/admin/garden.html`

### 板块管理（左侧）

- 点「+ 新增」创建板块（如"计算机知识"、"金融知识"）
- 点「×」删除板块（会同时删除板块内所有条目）

### 条目管理（右侧）

选中左侧板块后，右侧显示该板块的条目列表。

点「+ 添加条目」：
- **类型**：文章 / 视频 / 书籍 / 网站 / 账号（账号类型用于"我在哪里"板块）
- **标签**：逗号分隔，支持多标签，前台可按标签筛选
- **嵌入链接**：支持上传 PDF（点「⬆ 上传 PDF」），自动填入地址

## 主页配置

路径：`/admin/profile.html`

在这里配置主页展示的个人信息，修改后保存，刷新主页即可看到更新。

| 字段 | 说明 |
|---|---|
| 姓名 | 显示在 Hero 区域的名字 |
| 副标题 | 姓名下方的一行描述 |
| 个人简介 | 可选，显示在副标题下方 |
| 技能标签 | 彩色徽章，Go/Java/Python/Docker 有专属颜色，其他为灰色 |
| 经历时间线 | 每条含年份、职位/学校、补充说明 |
| 社交链接 | 含标签、URL、图标（推荐用 emoji），同时显示在 Hero 区和页面底部联系方式 |

---

## 修改管理员密码

通过环境变量设置，修改后重启服务生效：

```bash
ADMIN_USER=yourname ADMIN_PASS=strongpassword go run ./cmd/server
```

或在 `.env` 文件中设置（需自行实现 env 文件加载，或直接 export）。
